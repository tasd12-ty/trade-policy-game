# 两种初始化方法对比结果

数据: CN2017 vs US2017, 5部门, rho=0 (Cobb-Douglas)

## 结论

**两种方法求解的价格初值差异巨大（最大99.6%），不相等。**

| 指标 | 方法1 `initialize()` | 方法2 `initialize_from_equilibrium()` |
|------|---------------------|--------------------------------------|
| P=MC 零利润 | ~1e-13 (机器精度) | ~2-5% 偏差 |
| 商品市场出清 | ~1e-8 (机器精度) | ~0.1-5% 偏差 |
| 要素市场出清 | ~1e-9 (机器精度) | 48%-12700% 偏差 |
| 求解器信息 | 解析法+联合迭代 | scipy LM, converged=True, residual=1.83e-01 |

**方法1是正确的均衡解；方法2的scipy求解器收敛到了非均衡点（ftol满足但残差=0.18）。**

## 方法说明

### 方法1: `initialize()` (推荐)
1. 直接对数法解 P=MC: `(I-α_goods)·lnP = rhs` — 精确解
2. 值 Leontief 解产出: `(I-B)^{-1}·(PY_C + λ·PY_E)`
3. λ 出口缩放使要素市场出清
4. 外层迭代要素价格 w (锚 w₁=1)
5. 两国联合迭代 (10轮) 使进口价格一致

### 方法2: `initialize_from_equilibrium()`
1. ρ=0 解析解作初值 (equilibrium_rho0)
2. scipy least_squares (LM法) 求解完整非线性系统
3. 残差包含: 零利润 + 要素出清 + 商品出清 + 贸易收支 + 名义锚
4. 本次结果: converged=True, 但 residual=1.83e-01 >> 0
5. **ftol 终止但未真正收敛，陷入局部极小**
